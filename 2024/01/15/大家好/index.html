🚀 Hexo 博客搭建经历

为什么选择 Hexo？

  • 快速简单：基于 Node.js，生成静态页面速度极快
  • 主题丰富：有大量美观的主题可供选择
  • Markdown支持：专注于内容写作，无需关心样式

搭建过程中遇到的挑战

  1. 环境配置
    # 安装 Hexo
yard install -g hexo-cli

# 初始化博客
hexo init myblog
cd myblog

# 安装依赖
yard install
